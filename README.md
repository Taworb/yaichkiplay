---

## 📥 Как скачать

1. Откройте нужную папку в репозитории
2. Нажмите на файл
3. Нажмите кнопку **Download**

Или скачайте всё сразу: **Code → Download ZIP**

## ➕ Как добавить файл (для владельца)

### Через браузер

1. Зайдите в нужную папку
2. **Add file → Upload files**
3. Загрузите файлы → **Commit changes**

### Через Git

```bash
git clone https://github.com/YOUR-USERNAME/Voice.git
cd Voice

git add .
git commit -m "Добавлены новые файлы"
git push
```

> ⚠️ Максимальный размер файла на GitHub — **100 МБ**.  
> Для больших файлов используй [Git LFS](https://git-lfs.com/).
